# cola-react-ui

#### 一个基于React Hooks和TypeScript的组件库。

## Install

```shell
npm install cola-react-ui --save
```
## Quickstart
```tsx
import { Button } from 'cola-react-ui'
import 'cola-react-ui/dist/index.css'

const App = () => (
  <>
    <Button btnType="primary">PRESS ME</Button>
  </>
)
```
