# Menu组件
<br/>

#### 导航菜单，横向和纵向两种展示模式，可设置下拉子菜单。
<br/>

+ 横向菜单：

<Story name='horizontal' args={{}}>
  {args => 
  <Menu {...args}>
    <Menu.Item>Option 0</Menu.Item>
    <Menu.Item>Option 1</Menu.Item>
    <Menu.Item>Option 2</Menu.Item>
    <Menu.SubMenu title='Submenu'>
      <Menu.Item>Option 3</Menu.Item>
    <Menu.Item>Option 4</Menu.Item>
    </Menu.SubMenu>
  </Menu>}
</Story>

+ 纵向菜单：

<Story name='vertical' args={{
  style:{width: 300},
  mode: 'vertical'
}}>
  {args => 
  <Menu {...args}>
    <Menu.Item>Option 0</Menu.Item>
    <Menu.Item>Option 1</Menu.Item>
    <Menu.SubMenu title='Submenu'>
      <Menu.Item>Option 2</Menu.Item>
    <Menu.Item>Option 3</Menu.Item>
    </Menu.SubMenu>
  </Menu>}
</Story>

<br/>

#### 安装：

```sh
  npm install cola-comp --save
```
#### 基本用法：
```tsx
import Menu from './Menu'
<Menu 
  style={{width: 300}}
  mode={vertical}
  onSelect={(index)=>console.log(index)}
  >
  <Menu.Item>Option 0</Menu.Item>
  <Menu.Item>Option 1</Menu.Item>
  <Menu.Item>Option 2</Menu.Item>
  <Menu.SubMenu title='Submenu'>
    <Menu.Item>Option 3</Menu.Item>
  <Menu.Item>Option 4</Menu.Item>
  </Menu.SubMenu>
</Menu>
```
